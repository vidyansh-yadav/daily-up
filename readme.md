# Daily-Up - Developer Habit Tracker

A sophisticated habit tracking application with a developer/hacking theme interface.

## Features

- ✅ User Authentication (Login/Register)
- ✅ Habit Management (Create, Read, Update, Delete)
- ✅ Interactive Dashboard with Statistics
- ✅ Habit Calendar with Completion Tracking
- ✅ Time Table for Daily Schedule
- ✅ Progress Charts and Analytics
- ✅ Developer-themed UI with Hacker Aesthetics
- ✅ Daily Motivation Popups
- ✅ Skill Focus Tracking
- ✅ Progress Comparison (Day/Week/Month)

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB, Mongoose ODM
- **Charts:** Chart.js
- **Calendar:** FullCalendar.js
