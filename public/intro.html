Daily-Up ke Saare Features (Aasan Bhasha Mein)

📋 Habits (Aadatain)

Kya hai? Yeh woh kaam hain jo aap roz karna chahte ho.
Jaise: Coding, Exercise, Reading, Meditation
Example: "Mujhe roz 1 hour coding karni hai"
Aap habit add karo, Daily-Up yaad dilayega!

📅 Calendar (Kitaab)

Kya hai? Calendar mein aap dekh sakte ho ki kis din kaunsa habit kiya.
Jaise: Monday - Coding ✅, Tuesday - Exercise ✅
Example: "Monday ko coding ki thi? Calendar mein green tick dikhega!"
Jaise school attendance register!

🔥 Streaks (Lagataar)

Kya hai? Streak batata hai ki aapne lagataar kitne din habit kiya.
Jaise: 3 din coding ki → Streak = 3 🔥
Example: "Aaj coding ki, kal bhi ki, parson bhi ki → Streak 3!"
Break kiya to streak 0 se start!

⏰ Timetable (Program)

Kya hai? Yahan aap plan kar sakte ho ki kis time kaunsa habit karna hai.
Jaise: 9 AM Coding, 5 PM Exercise
Example: "Subah 9 baje coding, shaam 5 baje gym!"
Jaise school ka time table!

📊 Analytics (Report Card)

Kya hai? Analytics batata hai ki aap kitne acche ho.
Jaise: 80% habits complete kiye, 5 din streak
Example: "10 mein se 8 habit kiye = 80% success!"
Jaise school ka report card!

🍅 Pomodoro (Focus Timer)

Kya hai? Ek timer jo focus karne mein madad karta hai.
Jaise: 25 min padho, 5 min break
Example: "25 minute coding, phir 5 minute chai break!"
4 baar repeat karo, phir 15 min long break!

📔 Journal (Diary)

Kya hai? Yahan aap apne din ke baare mein likh sakte ho.
Jaise: Aaj mood kaisa tha? Kya seekha?
Example: "Aaj bahut maza aaya! Mood 10/10 😊"
Gratitude likho - aaj kiske liye thankful ho!

🏆 Challenges (Muqabla)

Kya hai? Doosron ke saath compete karo ya khud ko challenge do.
Jaise: 7 din coding challenge
Example: "7 din lagataar coding karo, inaam pao!"
Leaderboard pe apna rank dekho!

💬 Random Chat (Baatcheet)

Kya hai? Yahan aap doosre users se baat kar sakte ho.
Jaise: Kisi se bhi random chat karo!
Example: "Koi coding seekh raha hai? Baat karo!"
Admin special powers bhi hain!

🧠 Skills Focus (Hunar)

Kya hai? Yahan aap apni skills track kar sakte ho.
Jaise: JavaScript seekh rahe ho to progress dekho
Example: "JavaScript 65% complete, aur 35% baki!"
Har skill ka level dikhta hai!

XP aur Level kya hai?

• XP = Experience Points (Anubhav). Har habit complete karne par 50 XP milte hain!
• Level = 100 XP collect karne par level up! Level 1 → Level 2
• Badges = Kuch khaas kaam karne par (7 din streak) badge milta hai!
